# 舒尔特表训练工具 (Schulte Tables)

舒尔特表是一种通过在网格中快速定位数字或字母来训练注意力和视觉搜索能力的工具。本项目提供了一个功能丰富的舒尔特表在线训练平台，支持多种训练模式和自定义设置。

## 功能特点

- **多种表格尺寸**：支持自定义网格大小，满足不同训练需求
- **多样化训练模式**：包括正向计数、反向计数、发散计数等多种训练方式
- **实时统计**：记录训练时间、点击次数等数据，帮助用户追踪进步
- **鼠标轨迹可视化**：展示用户在训练过程中的鼠标移动路径，分析注意力分布
- **语言切换**：支持中英文界面切换，方便不同语言用户使用
- **丰富的视觉设置**：支持显示悬停效果、点击反馈、中心标记等多种视觉辅助功能

## 如何使用

1. 访问[项目网站]http://schulte.chenjiahui.cn  /或在本地运行
2. 在设置面板中选择合适的表格尺寸和训练模式
3. 点击"开始测试"按钮开始训练
4. 按照要求顺序点击数字完成训练
5. 查看统计结果和鼠标轨迹分析

## 本地部署

1. 克隆或下载本项目到本地
2. 使用任何Web服务器托管项目文件
3. 在浏览器中打开index.html即可使用

## 技术栈

- HTML5 + CSS3
- JavaScript
- Vue.js

## 开发背景

舒尔特表是一种被广泛应用于注意力训练的工具，本项目旨在提供一个易用、功能丰富的在线平台，帮助用户通过科学的方法提升注意力、视觉搜索速度和认知能力。

## 注意事项

- 建议每天进行短时间、多次数的训练，以获得最佳效果
- 训练时保持专注，避免分心
- 根据个人情况选择合适的难度级别，循序渐进地提升能力
